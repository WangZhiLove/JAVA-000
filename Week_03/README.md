Netty的核心就是 BECH：
- Bootstrap：启动线程，启动Socket，也就是程序的入口
- EventGroup：一般是两个，一个是Boos，一个是Work
- Channel
- Handler

都是知道有这么个东西，具体详细的就不是特别清除。

还有就是发展：事件处理机制 -> Reactor模型 -> Netty NIO，最后包括了Reactor单线程，非主从多线程
和主从多线程。每一个知识点都很深呀，得赶快构建自己的脑图了。
